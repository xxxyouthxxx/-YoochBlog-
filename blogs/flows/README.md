“心流”
> 将注意力完全投注在某活动的感觉
> <name>定义</name>

这种感觉就会在敲代码的时候还感觉不到，随着自己的任务逐渐完成，逐渐的感觉到自己的注意力开始分散，开始回想刚才的专注状态，这种感觉就是心流。

想要在一个很高的高度总结自己的心流和emotion，但非我说起真没有头绪，或许情绪就是这样的呢，没有开头和结尾，只能通过自己的感受来判断。来了也就走了，我想私自将这种感觉留存，就这样吧。
